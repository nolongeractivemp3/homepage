Hardware

Ich hoste es auf einem kleinen Raspberry Pi mit 4 GB RAM und einer 128-GB-MicroSD-Karte.

Software

Ich benutze das Standard Raspberry Pi OS und hoste fast alles über Docker (Compose). Ich hoste folgende Dienste in keinem bestimmten Reihenfolge:

  • Portainer für Management und so
  • Gitea weil GitHub stressig ist
  • Uptime Kuma für Monitoring
  • News 3001 selbst programmierter News scraper
  • File Browser für meine Dateien
  • Netdata für temperatur monitoring aber Netdata ist eig nicht so gut
  • Pocketbase als Datenbank für kleine Projekte
  • Casa OS ist eigentlich nicht so sicher deswegen enferne ich es langsam so das es wenigstens nicht mehr public ist
  • BlueskyScraper habe ich selbst programmiert um meine Bluesky Posts zu scrapen und in meine obsidian notes zu speichern

Networking

Ich benutze (leider) Wifi weil Kabel stressig sind. Portforwarding funktioniert auf dem Router nicht deswegen lasse ich alles über WireGuard an einen VPS von Oracle (Oracle ist schlecht ich weiß aber geht nicht anders und sie kriegen kein Geld von mir) und von da mit socat leiten damit es eine statische IP hat.
Die Domain habe ich von Hostinger und benutzt nginx für Routing und so.
