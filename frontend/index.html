Hardware

Ich hoste es auf einem kleinen Raspberry Pi mit 4 GB RAM und einer 128-GB-MicroSD-Karte.

Software

Ich benutze das Standard Raspberry Pi OS und hoste fast alles über Docker (Compose). Ich hoste folgende Dienste in keiner bestimmten Reihenfolge:

  • Portainer für Management und so
  • Gitea weil GitHub stressig ist
  • Uptime Kuma für Monitoring
  • News 3001, ein selbst programmierter News-Scraper
  • File Browser für meine Dateien
  • Netdata für Temperatur-Monitoring, aber Netdata ist eigentlich nicht so gut
  • Pocketbase als Datenbank für kleine Projekte
  • Casa OS ist eigentlich nicht so sicher, deswegen entferne ich es langsam, sodass es wenigstens nicht mehr public ist
  • BlueskyScraper habe ich selbst programmiert, um meine Bluesky-Posts zu scrapen und in meinen Obsidian-Notizen zu speichern

Networking

Ich benutze (leider) Wifi, weil Kabel stressig sind. Portforwarding funktioniert auf dem Router nicht, deswegen lasse ich alles über WireGuard an einen VPS von Oracle (Oracle ist schlecht ich weiß aber geht nicht anders und sie kriegen kein Geld von mir) und leite es von da mit socat weiter, damit es eine statische IP hat.
Die Domain habe ich von Hostinger und ich benutze nginx für Routing und so.
