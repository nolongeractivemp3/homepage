Self-hosting

Things I'm self-hosting

  • This site (obviously)
  • Gitea
  • Uptime Kuma
  • CasaOS to glue it all together
  • Portainer for tech help
  • Some simple services I made for automation
  • News 3001 (one of my projects)

The Stack

I'm using a Raspberry Pi 4 Model B (4GB RAM). All my services run on Docker with CasaOS as my home screen.

I'm routing everything through a free Oracle VPS to get a static public IP (my router sucks). Yes, yes, I know Oracle is bad, but I don't really have another choice. At least I'm not giving them money.
